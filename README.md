Description

This Python program analyzes a company's monthly sales data and provides key insights such as total sales, average sales, best and worst performing months, and months above or below average performance.

Features

Accepts monthly sales input (January to December)

Calculates:

Total yearly sales

Average monthly sales

Highest and lowest sales values

Identifies:

Best sales month

Worst sales month

Months with above-average sales

Months with below-average sales

Technologies Used

Python üêç

NumPy üì¶

How It Works

The program stores month names using a NumPy array.

User enters sales values (in $1000) for each month.

Sales data is converted into a NumPy array.

Statistical analysis is performed using NumPy functions.

Results are displayed in a clear, readable format.

Sample Output
--- Company Sales Analysis ---
Total Sales of the Year: 240 $
Average Monthly Sales: 20 $
Highest Sales: 35 $
Lowest Sales: 10 $

Best Month: Aug
Worst Month: Feb
Above Average Months: ['May' 'Jun' 'Jul' 'Aug']
Below Average Months: ['Jan' 'Feb' 'Mar']

How to Run
python sales_analysis.py

Input Format

Enter numeric values for each month

Sales values should be in $1000 units

Understanding NumPy arrays
